# homebrew-sparklines

Homebrew tap for [sparklines](https://github.com/deeplook/sparklines) — generate
sparkline charts for numbers using Unicode block characters only (e.g. `▃▁▄▁▅█▂▅`).

## Install

```sh
brew tap deeplook/sparklines
brew install sparklines
```

## Usage

```sh
sparklines 3 1 4 1 5 9 2 6
# → ▃▁▄▁▅█▂▅
```

## Upgrade

```sh
brew upgrade sparklines
```

## Uninstall

```sh
brew uninstall sparklines
brew untap deeplook/sparklines
```
